# **App Name**: CodeCanvas Portfolio

## Core Features:

- Profile Showcase: Display a professional profile including name, title, and a brief introduction.
- Project Gallery: Showcase a selection of featured projects with images and descriptions, each linking to its GitHub repository.
- Skills Overview: Highlight key technical skills and expertise areas, such as programming languages and frameworks.
- Contact Form: Enable visitors to directly send messages. Consider implementing a basic AI tool to filter spam from real inquiries.
- Footer Information: Include a standard footer with copyright information and links to professional profiles (e.g., LinkedIn).

## Style Guidelines:

- Primary color: Petrol blue (#005F73) — conveys confidence and depth, blending perfectly with dark tones for a professional yet modern look.
- Background color: Graphite gray (#1A1A1A) — provides a smooth dark base that enhances readability while maintaining a stylish and minimalist aesthetic.
- Accent color: Cyan blue (#0A9396) — adds a subtle pop of color for highlights, buttons, and interactive elements without breaking the overall dark harmony.
- Font pairing: 'Inter' for body text, a sans-serif offering excellent readability, and 'Space Grotesk' for headlines.
- Use clean and minimalist icons for navigation and skill representation.
- Employ a grid-based layout for balanced content distribution.
- Subtle transitions and hover effects for interactive elements, like button rollovers and project card expansions.
